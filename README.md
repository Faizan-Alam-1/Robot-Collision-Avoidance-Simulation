# Quickstart

### Step 1. Create virtual environment (recommended but not compulsory)
```bash
   python3 -m venv vir
   source vir/bin/activate
```    



### Step 2. install requirements.txt.
```bash
   pip install -r requirements.txt
```    


### Step 3. Run python script.
```bash
    python3 Final.py
```
